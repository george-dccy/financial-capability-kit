# 顶级银行人思维模型包

## 用途

帮助从业者在复杂业务中保持高质量判断和稳定执行，而不是只靠经验拍脑袋。

## 模块

- `decision-framework.md`
- `execution-rhythm.md`

## 常见问题

### 这个包会不会和具体产品知识冲突

不会。该包是"怎么思考、怎么推进"的方法层，不替代业务知识包。

## 来源

- 公开管理学与咨询方法常识（结构化问题分析、PDCA、风险管理通用框架）
- 金融从业公开培训中的流程管理与复盘方法

# 资产负债表

## 核心等式

**资产 = 负债 + 所有者权益**

这是资产负债表的基本逻辑。负债和所有者权益共同支撑了企业的资产规模。

## 银行视角的关键科目

### 资产侧

| 科目 | 银行关注点 |
|------|-----------|
| 货币资金 | 流动性直接反映偿债能力，但需区分受限资金 |
| 应收账款 | 账龄结构、周转率；与营业收入的比例关系 |
| 存货 | 变现能力、行业周期；跌价准备是否充分 |
| 固定资产 | 折旧政策；抵押价值 |
| 在建工程 | 转固时点影响利润和折旧 |

### 负债侧

| 科目 | 银行关注点 |
|------|-----------|
| 短期借款 | 银行融资占比；与货币资金的比例（偿债覆盖） |
| 应付账款 | 上下游占款能力；账期结构 |
| 应付债券 | 资本市场融资规模；评级信息 |
| 其他应付款 | 关联交易迹象；隐性负债信号 |

### 所有者权益

实收资本/股本、资本公积、盈余公积、未分配利润是常见科目。银行关注所有者权益的稳定性——增资和分红历史反映股东支持力度。

## 专业判断提示

1. **资产质量比规模重要**：虚增资产（应收账款异常、存货积压）是常见报表粉饰手段
2. **负债结构比总量重要**：短借长用是流动性风险的典型特征
3. **关联往来要细看**：其他应收/应付款大额且长期未清，可能是隐性负债或资金占用

## 与银行信审的连接

银行在分析资产负债表时，通常计算：
- 资产负债率（杠杆水平）
- 流动比率 + 速动比率（短期偿债能力）
- 应收账款周转天数、存货周转天数（经营效率）

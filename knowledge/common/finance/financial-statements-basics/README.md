# 财报基础知识包

## 用途

帮助银行对公客户经理、风险管理者等，在客户接触、信用分析、需求判断等场景中，建立对财务报表的专业理解。

三大报表是银行人与企业对话的基础语言。理解财报结构，才能问出正确的问题。

## 与 reference skill 的分层

- **本包**：三大报表的专业知识支撑层，回答「这是什么」「怎么看」
- **reference skill**（如 `enterprise-cash-flow-judgment`）：回答「用什么框架判断」「优先看什么」

本包不提供分析框架，只提供可信赖的事实基础。

## 模块

- `balance-sheet.md` — 资产负债表
- `income-statement.md` — 利润表
- `cash-flow-statement.md` — 现金流量表

## 边界提示

- 不提供税务处理细节
- 不提供具体行业估值方法
- 不提供最新监管政策的即时解读

## 常见问题

### 这个包是否提供企业信用评级

不提供。本包只提供财报基础知识层面的理解框架，不包含银行内部评级模型或外部评级机构方法论。

### 这个包是否能帮助判断具体授信额度

不能。授信额度涉及银行内部政策、客户贡献度、综合收益测算等复杂因素，需要结合具体客户关系和银行信贷政策综合判断。

### 是否覆盖财务报表粉饰的识别方法

本包涉及部分识别思路（如关注异常科目、非经常性损益比例等），但不系统覆盖财务舞弊识别的专业方法。如需深入，建议参考中国注册会计师协会发布的审计准则相关资料。

### 三大报表的分析是否有优先顺序

有。银行视角的推荐顺序：

1. **现金流量表**（经营现金流净额）—— 验证企业是否真的在赚钱
2. **资产负债表**（结构与科目异常）—— 看家底和杠杆
3. **利润表**（盈利质量）—— 验证持续性

### 能否用于跨境企业或上市公司财报分析

基本框架通用。但上市公司还需关注合并报表与母公司报表的差异、股份支付、公允价值计量等复杂会计处理，以及监管披露要求。

### 是否提供行业对比基准数据

不提供。本包提供分析框架和科目理解，不包含行业对比基准数据。如需行业对比，应参考行业协会数据或Wind、Choice等金融数据终端。

## 来源

### 会计准则与监管框架

- 《企业会计准则》—— 财政部发布，资产负债表、利润表、现金流量表的基本确认与计量规则
- 《企业会计准则第31号——现金流量表》
- 《企业会计准则第30号——财务报表列报》
- 中国证券监督管理委员会《公开发行证券的公司信息披露编报规则》

### 财报分析框架

- 中国注册会计师协会（CICPA）《审计》教材—— 财务报告分析部分
- 中国人民银行《金融机构企业信用评级管理指引》
- 国务院国有资产监督管理委员会《企业绩效评价标准值》

### 银行信审视角

- 国有商业银行对公信贷业务公开培训资料（各行公开渠道）
- 中国银行业协会相关公开出版物

### 使用说明

以上来源均为公开发布的权威资料。部分来源需要在联网环境下通过官方渠道核验具体条款编号和最新修订状态。

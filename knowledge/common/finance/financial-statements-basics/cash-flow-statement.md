# 现金流量表

## 核心等式

**经营现金流 + 投资现金流 + 融资现金流 = 现金净流量**

现金流量表反映企业真实资金流转情况，是对资产负债表和利润表的有力校验。

## 银行视角的关键科目

### 经营活动现金流（核心关注）

| 科目 | 银行关注点 |
|------|-----------|
| 销售商品、提供劳务收到的现金 | 与营业收入的比例（收入含金量） |
| 购买商品、接受劳务支付的现金 | 与营业成本的比例（采购付款管理） |
| 支付给职工的现金 | 员工规模变化印证业务扩张/收缩 |
| 收到的税费返还 | 政策补贴信号 |
| 经营现金流净额 | **最核心指标**：正值且稳定代表主业自我造血能力强 |

### 投资活动现金流

| 科目 | 银行关注点 |
|------|-----------|
| 收回投资收到的现金 | 是否在收缩投资战线 |
| 取得投资收益收到的现金 | 与利润表投资收益是否匹配 |
| 购建固定资产、无形资产支付的现金 | 资本开支强度；扩张还是维持 |
| 投资支付的现金 | 是否在多元化扩张；风险信号 |

### 融资活动现金流

| 科目 | 银行关注点 |
|------|-----------|
| 吸收投资收到的现金 | 股权融资能力；股东支持信号 |
| 取得借款收到的现金 | 银行融资规模变化 |
| 偿还债务支付的现金 | 到期债务偿还压力 |
| 分配股利、利润支付的现金 | 分红力度；股东回报预期 |

## 专业判断提示

1. **经营现金流是信用分析的生命线**：经营现金流持续为负，说明企业靠融资维持生存，是重大风险信号
2. **收入含金量 = 经营现金流/营业收入**：这个比值长期低于0.8，说明收入质量有问题（赊销过度、寅吃卯粮）
3. **投资现金流负数不一定是坏事**：主动扩张性资本开支代表对未来有信心；但要与融资现金流配合看——借债投资是激进信号
4. **三张表要对照看**：利润盈利但经营现金流持续为负，典型的「报表好看但不赚钱」

## 与银行信审的连接

银行在分析现金流量表时，通常计算：
- 经营现金流/带息负债（偿债覆盖）
- 经营现金流/营业收入（收入含金量）
- 资本开支/经营现金流（自由现金流留存）
- 融资前后的债务结构变化

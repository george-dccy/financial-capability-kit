# 利润表

## 核心等式

**净利润 = 营业收入 − 营业成本 − 费用 − 税金 + 营业外收支 − 所得税**

利润表反映企业在一定期间的经营成果。

## 银行视角的关键科目

### 收入侧

| 科目 | 银行关注点 |
|------|-----------|
| 营业收入 | 增长趋势；与行业增速是否匹配；是否依赖单一客户或产品 |
| 营业外收入 | 非经常性损益占比过高，说明主业不强 |
| 投资收益 | 是否为一次性；与主业的关联度 |

### 成本费用侧

| 科目 | 银行关注点 |
|------|-----------|
| 营业成本 | 与收入的配比关系；原材料价格波动传导 |
| 销售费用、管理费用 | 费用率趋势；费用与收入的匹配度 |
| 财务费用 | 有息负债规模的风向标；与营业收入的比例反映融资成本负担 |
| 资产减值损失 | 坏账计提是否充分；是否存在利润操纵空间 |

### 最终结果

| 科目 | 银行关注点 |
|------|-----------|
| 营业利润 | 反映主业盈利能力，是信审最看重的指标之一 |
| 净利润 | 需关注是否含非经常性损益；历史趋势比单年数更重要 |

## 专业判断提示

1. **收入质量比增速重要**：增长是否伴随现金回流（对照现金流量表）；增速是否异常高于行业
2. **毛利率是竞争力的直接体现**：毛利率持续下降可能说明产品议价能力减弱或行业竞争加剧
3. **非经常性损益是调节利润的常见手段**：政府补助、投资处置收益等一次性利得，不能视为持续盈利能力
4. **财务费用是融资需求的体温计**：财务费用占收入比持续上升，说明有息负债在累积

## 与银行信审的连接

银行在分析利润表时，通常计算：
- 毛利率、净利率（盈利能力）
- 营业收入增长率（成长性）
- 财务费用/营业收入（融资成本负担）
- EBITDA（利息税前利润，用于偿债能力测算）

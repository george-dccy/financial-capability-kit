# 线上授信自动化审批方案基础提纲包

## 用途

帮助用户和 Agent 在起草线上授信自动化审批业务方案时，先对齐：

- 该方案通常要覆盖哪些模块
- 哪些信息必须补
- 哪些边界必须单独提示

它不替代内部制度、审批规则或最终授信方案。

## 模块

- `outline.md`
- `admission-and-risk-gates.md`
- `management-and-consumer-protection.md`

## 常见问题

### 这个包是否能直接给出授信审批结论

不能。它只提供公开、稳定、可引用的提纲和补数框架。

### 这个包是否包含内部制度原文或准入阈值

不包含。涉及内部政策和阈值时，应明确写待人工确认。

### 这个包更适合哪些场景

当前更适合对公、核心企业上下游、平台交易、供应链和场景金融相关的线上化授信方案草拟。

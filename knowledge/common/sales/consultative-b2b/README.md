# 顾问式B2B销售技巧包

## 用途

帮助银行人在客户沟通中先洞察需求、再提出方案，提升营销命中率和客户信任度。

## 模块

- `discovery-questions.md`
- `objection-handling.md`

## 常见问题

### 这个包会不会让回答过度销售

不会。该包强调顾问式沟通和价值澄清，不鼓励强推销。

## 来源

- 公开 B2B 顾问式销售方法常识（需求洞察、价值表达、异议处理）
- 企业客户沟通的公开实践模型

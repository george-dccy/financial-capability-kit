# 与已有知识包的关联

## 概述

光大银行在供应链金融领域已有两个专项知识包，分别覆盖 e付通和电费通两个具体场景产品。本模块说明它们与阳光供应链产品族的关系，以及如何配合使用。

---

## e付通 知识包

### 基本信息

- **ID**: knowledge.banks.ceb.transaction-banking.e-fu-tong
- **路径**: knowledge/banks/ceb/transaction-banking/e-fu-tong/
- **定位**: 核心企业-供应商场景下的 1+N 保理线上化模式

### 与阳光供应链的关系

e付通 与以下供应链能力相关：

1. **1+N 保理相关产品** — e付通 的核心是 1+N 保理模式，由核心企业授信支持供应商融资
2. **阳光供应链云平台** — 对客服务平台/线上化渠道能力，支持供应链业务线上化承载

因此，e付通 可以理解为 1+N 保理场景与供应链线上化承载能力在核心企业-供应商场景下的具体产品表达。

### 何时使用 e付通 知识包

- 客户提到核心企业-供应商融资场景
- 客户关心 1+N 保理、应收账款转让
- 客户需要供应链线上化服务

### 知识包内容

- 产品定位与公开依据
- 销售发现性问题
- 客户常见问题

## 电费通 知识包

### 基本信息

- **ID**: knowledge.banks.ceb.trade-finance.dian-fei-tong
- **路径**: knowledge/banks/ceb/trade-finance/dian-fei-tong/
- **定位**: 电费缴纳场景下的国内证+福费廷组合

### 与阳光供应链的关系

电费通 建立在国内信用证和福费廷产品基础上，是贸易融资产品在电费缴纳行业场景的具体应用。

### 何时使用 电费通 知识包

- 客户提到电费缴纳、电力行业供应链
- 客户关心国内信用证、福费廷
- 客户需要电费结算融资方案

### 知识包内容

- 产品结构与场景说明
- 知识引用与来源
- 常见问题

---

## 关联使用建议

### 场景判断流程

```
客户提到供应链金融
    ↓
判断贸易类型
    ├─ 国内贸易
    │   ├─ 核心企业-供应商 → 查看 e付通 知识包
    │   ├─ 电费缴纳场景 → 查看 电费通 知识包
    │   └─ 其他 → 回到产品目录核验保理、订单贷、保兑仓等方向
    ├─ 跨境贸易
    │   ├─ 自贸区企业 → 回到产品目录核验跨境资金池、结售汇等方向
    │   └─ 一般跨境 → 回到产品目录核验跨境汇款、托收、信用证等方向
    └─ 担保需求 → 阳光保函
```

### 注意事项

- e付通 和 电费通 是阳光供应链能力的具体场景表达，不是独立于供应链的全新产品
- 首次沟通时，先判断客户场景再决定引用哪个知识包
- 如果客户场景不明确，先用本包的 `product-family.md` 和 `scenarios.md` 做整体梳理

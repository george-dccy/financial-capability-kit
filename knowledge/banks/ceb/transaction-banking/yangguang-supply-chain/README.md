# 光大银行-阳光供应链知识包

## 适用问题

- 阳光供应链包含哪些子产品，它们分别是什么
- 阳光资财、阳光保理、阳光供应链云平台分别适合什么场景
- e付通和电费通与阳光供应链是什么关系
- 阳光自贸通和阳光国际结算的定位有什么区别
- 首次沟通供应链金融业务应先判断什么

## 模块

- `modules/product-family.md` — 阳光供应链产品族总览与各子产品定位
- `modules/scenarios.md` — 场景矩阵与产品匹配
- `modules/existing-packs.md` — 与已有 e付通、电费通知识包的关联
- `faq.md`
- `sources.md`

## 使用建议（给 Agent）

1. 先读 `product-family`，理解阳光供应链六大子产品的整体架构。
2. 再读 `scenarios`，根据客户场景匹配合适的产品组合。
3. 需要了解已有知识包时读 `existing-packs`，获取 e付通和电费通的详细信息路径。
4. 客户问答阶段优先读 `faq.md`，保持结论稳妥。
5. 如果需要把知识转成会谈提纲或推进动作，补读 `skills/action/market-corporate-client`。

## 边界

该包只支持公开知识解释与产品体系理解，不输出审批、授信、费率、时效、系统对接能力或上线承诺。

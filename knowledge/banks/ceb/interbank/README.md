# 光大银行-同业金融目录地图

日期：`2026-04-21`

## 定位

本目录用于识别光大银行同业金融与托管业务的产品族、平台和场景，不输出费率、审批或授信承诺。

## 目录结构

```
interbank/
├── README.md                    # 本文件：目录总览
├── interbank-business-map/     # 同业金融业务地图（货币市场、票据、投行）
├── custody-map/                # 托管业务地图（公募基金、保险、银行理财、信托、年金等）
└── interbank-finance-basics/   # 同业金融基础知识包（选建）
```

## 1. 同业金融业务（产品族）

来源：光大银行公司业务页

### 1.1 货币市场业务

| 产品/服务 | 类型 | 说明 |
|---------|------|------|
| 同业存款 | 产品 | 金融机构间资金存放 |
| 同业借款 | 产品 | 金融机构间资金融通 |
| 同业存单 | 产品 | NCD，记账式定期存款凭证 |
| 阳光"财汇盈" | 产品 | 同业金融解决方案品牌 |

### 1.2 票据业务

| 产品/服务 | 类型 | 说明 |
|---------|------|------|
| 商业汇票回购 | 产品 | 票据融资类 |
| 商业汇票转贴现 | 产品 | 票据二级市场交易 |
| 阳光 ABS | 产品/平台 | 资产证券化相关 |

### 1.3 跨境与外汇

| 产品/服务 | 类型 | 说明 |
|---------|------|------|
| 代客外汇 | 产品 | 外汇交易相关 |
| QDII/QFII 托管 | 托管+通道 | 跨境投资服务 |

### 1.4 银期合作

| 产品/服务 | 类型 | 说明 |
|---------|------|------|
| 阳光银期通 | 平台/产品 | 期货公司资金划转通道 |

| 分类 | 同业金融例子 |

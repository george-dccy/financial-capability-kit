# 光大银行-云缴费平台目录地图

日期：`2026-04-27`

## 定位

本目录用于识别光大云缴费平台的缴费场景类别、平台能力、接入渠道，及其与统一支付、综合收单的关系。不输出费率、系统对接方案或业务办理承诺。

## 目录结构

```
cloud-payment-map/
├── README.md                      # 本文件：目录总览
├── modules/
│   ├── scenario-overview.md       # 缴费场景总览与分类
│   ├── platform-capabilities.md   # 平台能力说明
│   └── channel-output.md          # 渠道输出与接入方式
├── faq.md                         # 常见问题
└── sources.md                     # 来源记录
```

## 适用问题

- 光大云缴费覆盖哪些缴费场景
- 云缴费与统一支付、综合收单是什么关系
- 云缴费平台接入过哪些输出渠道
- 首次沟通云缴费业务应先了解什么

## 模块

- `modules/scenario-overview.md` — 缴费场景总览（水电煤、社保、公积金、交通等）
- `modules/platform-capabilities.md` — 平台能力与统一支付、综合收单的关系
- `modules/channel-output.md` — 渠道输出与接入方式
- `faq.md` — 常见问题
- `sources.md` — 来源记录

## 边界

本目录仅支持公开知识解释，不输出具体费率、系统对接方案或业务办理承诺。

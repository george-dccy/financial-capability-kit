# 缴费场景总览

## 云缴费平台定位

光大云缴费是光大银行建设的数字便民缴费平台，旨在服务百姓日常生活，助力各地数字政务服务能力提升，实现"信息多跑路，群众少跑腿"。

## 缴费场景类别

光大云缴费覆盖的缴费场景分为以下大类：

| 场景类别 | 具体内容 | 说明 |
|---------|---------|------|
| 公用事业 | 水、电、煤 | 居民日常公用事业缴费 |
| 社会保障 | 社保 | 社会保险缴纳 |
| 住房保障 | 公积金 | 住房公积金缴存 |
| 交通出行 | 交通罚款、ETC等 | 交通相关费用 |
| 政府非税 | 非税收缴 | 中央及地方非税收入缴款 |
| 教育医疗 | 学费、医疗费等 | 公共服务领域 |
| 社区生活 | 物业费等 | 社区生活服务 |

## 平台规模（2023年数据）

- 接入缴费代收项目：**超1.6万余项**
- 年缴费笔数：**超28亿笔**
- 年缴费金额：**超8000亿元**
- 输出渠道接入：**815家**

## 与统一支付、综合收单的关系

```
光大数字金融平台层
├── 云缴费平台    → 面向缴费场景的专业平台（民生缴费、政府非税等）
├── 统一支付平台  → 企业级支付能力聚合（聚合支付服务）
├── 综合收单平台  → 收单业务整合（商户收款）
└── 资金监管平台  → 客户资金监管服务
```

关系说明：

| 平台 | 定位 | 与云缴费的关系 |
|------|------|-------------|
| 云缴费 | 缴费场景开放平台 | 专注民生缴费场景，是光大便民服务的核心平台 |
| 统一支付 | 聚合支付服务 | 输出支付能力，云缴费的部分场景通过统一支付完成资金结算 |
| 综合收单 | 收单业务整合 | 服务商户收款，与云缴费面向的场景不同 |
| 资金监管 | 资金监管服务 | 可与云缴费联动，支持政府非税等场景的监管要求 |

## 使用边界

- 水电煤、社保公积金等场景类别为公开报道的分类
- 具体费率、接入条件、系统对接方案不在本目录范围内
- 不构成业务办理承诺

# 平台能力说明

## 云缴费核心能力

光大云缴费作为数字便民服务的基础平台，具备以下核心能力：

### 1. 缴费代收能力

| 能力 | 说明 |
|------|------|
| 多场景覆盖 | 支持水电煤、社保、公积金、交通罚款等多种缴费场景 |
| 标准化接入 | 为收费单位提供标准化的缴费接入接口 |
| 实时/批量子缴 | 支持实时缴费和批量代扣 |

### 2. 渠道输出能力

| 能力 | 说明 |
|------|------|
| 多平台输出 | 缴费服务输出至微信、支付宝、华为、数字人民币APP等主流平台 |
| 开放API | 为合作伙伴提供标准API接入 |

### 3. 数字政务联动能力

| 能力 | 说明 |
|------|------|
| 政务对接 | 助力各地数字政务服务能力提升 |
| 非税缴款 | 支持中央及地方非税收入电子缴款 |

## 四大企业级平台协同

光大银行围绕"存、贷、汇、管"数字化基础能力，优化以下四大企业级平台：

| 平台 | 定位 | 与云缴费的关系 |
|------|------|-------------|
| 统一支付 | 聚合支付服务 | 云缴费的资金结算通道 |
| 综合收单 | 收单业务整合 | 与云缴费场景互补 |
| 资金监管 | 客户资金监管 | 云缴费部分场景的资金监管支撑 |
| 开放银行 | API输出 | 云缴费对外输出的技术通道 |

## 能力边界

- 本节描述基于公开报道的平台能力定位
- 具体接口文档、系统对接方案不在本目录范围内
- 不构成技术对接承诺或费率说明

# 渠道输出与接入方式

## 主要输出渠道

光大云缴费已接入以下类型渠道（截至2023年数据）：

| 渠道类型 | 代表渠道 | 说明 |
|---------|---------|------|
| 移动支付 | 微信支付、支付宝 | 主流第三方支付平台 |
| 硬件生态 | 华为（鸿蒙等） | 手机厂商内置支付 |
| 数字货币 | 数字人民币APP | 央行数字货币试点 |
| 银行渠道 | 光大手机银行、企业网银 | 自有渠道 |
| 政务平台 | 各省市政务服务平台 | 数字政务入口 |

## 接入渠道规模

| 指标 | 数据（2023年） |
|------|--------------|
| 输出渠道总数 | 815家 |
| 覆盖缴费项目 | 超1.6万余项 |
| 年缴费笔数 | 超28亿笔 |
| 年缴费金额 | 超8000亿元 |

## 渠道接入模式

```
收费单位（如水务、电力、社保局等）
    ↓  标准化接口
光大云缴费平台
    ↓  聚合输出
微信 / 支付宝 / 华为 / 数字人民币APP / 政务平台 / 自有银行渠道
    ↓
缴费用户
```

## 与统一支付的关系

| 对比维度 | 云缴费 | 统一支付 |
|---------|-------|---------|
| 核心功能 | 缴费场景聚合 | 支付能力聚合 |
| 服务对象 | 居民、政府部门、收费单位 | 企业商户 |
| 典型场景 | 水电煤缴费、社保缴纳 | 线上线下收款 |

## 使用边界

- 渠道接入条件和技术规范不在本目录范围内
- 具体渠道费率由各方协议约定
- 不构成渠道接入承诺

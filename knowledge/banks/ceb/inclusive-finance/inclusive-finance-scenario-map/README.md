# 光大银行-普惠金融场景地图知识包

## 适用问题

- 光大银行普惠金融包含哪些产品系列
- 阳光普惠和阳光兴农分别覆盖什么客群
- 灵工通、安居通等场景产品是什么
- 普惠金融的政策背景和发展方向是什么
- 首次沟通普惠金融业务应先判断什么

## 模块

- `modules/scenario-overview.md` — 普惠金融场景总览与产品分类
- `modules/key-products.md` — 重点产品详解（阳光普惠、阳光兴农、场景产品）
- `modules/policy-background.md` — 政策背景与监管要求
- `faq.md`
- `sources.md`

## 使用建议（给 Agent）

1. 先读 `scenario-overview`，理解普惠金融的整体产品架构和场景分类。
2. 再读 `key-products`，掌握阳光普惠三类产品、阳光兴农四类产品和重点场景产品。
3. 需要政策背景时读 `policy-background`，了解"五篇大文章"等监管要求。
4. 客户问答阶段优先读 `faq.md`，保持结论稳妥。
5. 如果需要把知识转成会谈提纲或推进动作，补读 `skills/action/market-corporate-client`。

## 边界

该包只支持公开知识解释与产品体系理解，不输出审批、授信、费率、时效或系统对接承诺。

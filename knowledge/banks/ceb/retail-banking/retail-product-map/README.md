# 光大银行-零售产品谱系地图知识包

## 适用问题

- 光大银行零售业务包含哪些产品线
- 个人贷款有哪些产品，分别适合什么场景
- 出国金融3.0和线上存款证明是什么
- 阳光借记卡和信用卡有什么特色功能
- 网络贷款产品有哪些

## 模块

- `modules/product-spectrum.md` — 零售产品谱系总览与七大产品线
- `modules/key-categories.md` — 重点品类详解（个人贷款、出国金融、贵金属）
- `modules/digital-channels.md` — 数字渠道与线上化服务
- `faq.md`
- `sources.md`

## 使用建议（给 Agent）

1. 先读 `product-spectrum`，理解零售业务七大产品线的整体架构。
2. 再读 `key-categories`，掌握个人贷款、出国金融、贵金属等重点品类。
3. 需要了解线上化服务时读 `digital-channels`，掌握手机银行、阳光普惠云等数字渠道。
4. 客户问答阶段优先读 `faq.md`，保持结论稳妥。
5. 如果需要把知识转成会谈提纲或推进动作，补读 `skills/action/market-corporate-client`。

## 边界

该包只支持公开知识解释与产品体系理解，不输出审批、授信、费率、时效或系统对接承诺。

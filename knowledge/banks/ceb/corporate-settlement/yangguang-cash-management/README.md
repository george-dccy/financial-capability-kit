# 光大银行-阳光金管家现金管理知识包

## 适用问题

- 阳光金管家是什么，它和现金管理业务是什么关系
- 标准版、直通式、跨行通、云财资四个子产品分别适合什么场景
- 现金管理六大功能模块分别覆盖哪些企业财资管理需求
- 云财资和标准版的核心区别是什么
- 企业客户首次沟通现金管理业务应先判断什么

## 模块

- `modules/overview.md` — 阳光金管家品牌概述与产品体系架构
- `modules/sub-products.md` — 四大子产品矩阵与适用场景
- `modules/functions.md` — 六大功能模块详解
- `faq.md`
- `sources.md`

## 使用建议（给 Agent）

1. 先读 `overview`，理解"一个平台、四个渠道、六大功能"的整体架构。
2. 再读 `sub-products`，区分标准版/直通式/跨行通/云财资的定位差异和适用客群。
3. 需要功能细节时读 `functions`，了解账户管理、收付款、流动性、融资、预算、风控六大模块。
4. 客户问答阶段优先读 `faq.md`，保持结论稳妥，不把公开描述说成内部承诺。
5. 如果需要把知识转成会谈提纲或推进动作，补读 `skills/action/market-corporate-client`。

## 边界

该包只支持公开知识解释与产品体系理解，不输出审批、授信、费率、时效、系统对接能力或上线承诺。

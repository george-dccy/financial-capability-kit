1. `transaction-banking/product-catalog`

2. `corporate-settlement/basic-settlement`
   - 对公基础结算与账户收付
3. `transaction-banking/e-fu-tong`
   - e付通及核心企业-供应商协同场景
4. `trade-finance/dian-fei-tong`
   - 电费通 / 电费证及电费缴纳场景结构理解

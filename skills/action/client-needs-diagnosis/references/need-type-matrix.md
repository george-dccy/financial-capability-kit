# 需求类型矩阵

## 需求类型一级分类

| 一级分类 | 说明 | 典型业务场景 |
|---------|------|------------|
| 结算与现金管理 | 日常收付款、资金归集、账务管理 | 收款、付款、对账、资金归集、工资代发 |
| 贸易融资 | 采购垫资、销售回款保障、信用证、供应链账期 | 采购垫资、订单融资、信用证、供应链账期 |
| 供应链金融 | 核心企业上下游融资、确权、担保 | 核心企业确权、经销商融资、担保融资 |
| 普惠金融 | 小微企业贷款、税务贷款、场景贷款 | 税务贷款、发票贷款、场景白名单 |
| 信贷融资 | 固定资产贷款、流动资金贷款、项目融资 | 扩产投资、项目交付、流动资金补充 |
| 外汇与跨境 | 结售汇、跨境结算、保函、汇率避险 | 出口收款、进口付汇、汇率锁定 |
| 数字化转型 | 银企直连、API 接口、电子票据、供应链平台 | 银企直连、财务系统对接、电子票据 |

## 二级细分与信号对照

### 结算与现金管理

| 二级细分 | 典型信号 |
|---------|---------|
| 基础结算 | 收款难、对账繁琐、跑银行多 |
| 现金管理 | 资金分散、归集需求、现金头寸管理 |
| 企业收单 | 收款渠道、POS、线上收款 |
| 工资代发 | 工资发放、报销款 |

### 贸易融资

| 二级细分 | 典型信号 |
|---------|---------|
| 信用证 | 进出口信用证、跟单信用证 |
| 托收 | 出口托收、进口托收 |
| 保函 | 投标保函、履约保函、付款保函 |
| 订单融资 | 订单在手、备货资金 |
| 发票融资 | 应收账款、发票垫资 |

### 供应链金融

| 二级细分 | 典型信号 |
|---------|---------|
| 核心企业确权 | 核心企业承诺付款、应付账款 |
| 经销商融资 | 下游经销商资金压力 |
| 担保融资 | 核心企业担保、联合担保 |

### 普惠金融

| 二级细分 | 典型信号 |
|---------|---------|
| 税务贷款 | 纳税多、税务记录良好 |
| 发票贷款 | 发票多、应收账款 |
| 场景白名单 | 特定场景（如电商、物流） |
| 政策性贷款 | 符合政策导向、普惠指标 |

### 信贷融资

| 二级细分 | 典型信号 |
|---------|---------|
| 流动资金贷款 | 短期周转、补充流动资金 |
| 固定资产贷款 | 购建资产、扩产投资 |
| 项目融资 | 项目交付周期、项目贷款 |
| 经营性物业贷款 | 物业经营、租金收入 |

### 外汇与跨境

| 二级细分 | 典型信号 |
|---------|---------|
| 结售汇 | 出口收汇、进口付汇 |
| 汇率避险 | 汇率波动、锁汇需求 |
| 跨境人民币 | 跨境结算、人民币国际化 |
| 保函 | 跨境保函、备用信用证 |

### 数字化转型

| 二级细分 | 典型信号 |
|---------|---------|
| 银企直连 | 财务系统对接、自动转账 |
| API 开放银行 | 系统集成、API 接口 |
| 电子票据 | 电子票据、供应链票据 |
| 供应链平台 | 供应链数字化、平台确权 |

## 行业与需求类型对照参考

| 行业 | 高概率需求类型 |
|------|--------------|
| 制造业 | 贸易融资、供应链金融、固定资产贷款 |
| 商贸批发 | 发票融资、流动资金贷款、结算管理 |
| 进出口 | 外汇与跨境、信用证、贸易融资 |
| 工程建设 | 保函、固定资产贷款、项目融资 |
| 科技/互联网 | 数字化转型、股权融资、流动资金 |
| 连锁零售 | 结算管理、供应链金融、普惠金融 |
| 中小微企业 | 普惠金融、税务贷款、流动资金 |

# Frameworks

## 一、银行人读财报的顺序

### 第一步：先看现金流量表（不是利润表）

银行最关注现金，因为"纸上富贵"救不了到期贷款。

**为什么要先看现金流量表：**

1. 经营现金流是否為正
   - 正：主营业务能自我造血
   - 负：不一定是坏事，但必须追问原因（行业特性？季节性？还是真失血？）

2. 经营现金流与净利润的差距
   - 差距大 → 利润可能来自赊销堆积或会计调节
   - 差距小 → 利润质量高，现金转化能力强

3. 三段现金流的结构关系
   - 经营正 + 投资负 + 融资负：成熟稳健型
   - 经营正 + 投资负 + 融资正：扩张型，需要判断融资用途
   - 经营负 + 投资负 + 融资正：高度关注型，靠输血维持

### 第二步：再看资产负债表

**核心关注点：**

1. 资产结构
   - 货币资金多寡：直接反映短期偿债能力
   - 应收账款和存货：是否堆积？是否在恶化？
   - 固定资产占比：资本密集度，决定行业特性

2. 负债结构
   - 短期负债占比：越高，流动性压力越大
   - 有息负债vs无息负债：银行借款 vs 供应商欠款性质不同
   - 负债率变化趋势：是在加杠杆还是去杠杆

3. 所有者权益
   - 注册资本是否实缴
   - 未分配利润是否可观
   - 历年分红情况

### 第三步：最后看利润表

利润表是用来"验证"的，不是用来"判断"的。

**验证逻辑：**

1. 收入增长是否有现金流支撑
2. 利润增长是否伴随着资产扩张（否则可能是会计处理）
3. 三项费用的变化是否合理
4. 毛利率是否稳定（大幅波动需要解释）

## 二、关键比率的银行视角

| 比率 | 计算方式 | 银行关注逻辑 |
|------|----------|-------------|
| 经营现金流/总债务 | 经营现金流 ÷ 全部有息负债 | 现有债务靠经营现金还款的能力 |
| 流动比率 | 流动资产 ÷ 流动负债 | 短期偿债能力参考 |
| 应收账款周转天数 | 360 ÷ (营收÷应收账款) | 回款质量，高于行业需追问 |
| 存货周转天数 | 360 ÷ (成本÷存货) | 存货变现能力 |
| 现金比率 | 货币资金 ÷ 流动负债 | 最保守的短期偿债指标 |

**注意：** 单一指标没有意义，必须结合行业特性和企业具体阶段判断。

## 三、行业特性对财报的影响

不同行业的财务逻辑差异巨大，不能用统一标准：

| 行业 | 财报特征 | 银行关注重点 |
|------|----------|-------------|
| 制造业 | 固定资产重、存货周转慢 | 产能利用率、存货跌价准备 |
| 贸易业 | 应收账款大、周转快 | 客户集中度、回款账龄 |
| 建筑业 | 垫资施工、现金流差 | 项目回款节点、甲方付款能力 |
| 房地产 | 资金密集、杠杆高 | 销售回款、在建项目进度 |
| 服务业 | 人工成本高、轻资产 | 人工成本占比、客户稳定性 |

## 四、判断顺序口诀

1. **先看钱，不看利润** — 现金流是偿还能力的直接证据
2. **再看结构，不看总量** — 资产负债结构揭示真正的风险点
3. **后验利润，不先验** — 利润表用来验证，不是用来决策
4. **行业对标，不统一** — 不同行业用不同尺子量
5. **三年趋势，不一年** — 单一年份数据容易失真

---

## 五、使用边界

- 三张报表必须交叉验证，单看一张会失真
- 没有三年数据，趋势判断不可靠
- 比率必须结合行业平均，不是越高越好
- 非上市企业的财报可信度需要更多渠道验证
- 财报只是起点，还需结合经营、担保、征信等综合判断

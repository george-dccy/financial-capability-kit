# Examples

## 示例 1：制造业客户 — 稳定型

**现象：**

- 近三年经营现金流：+2000万、+2500万、+2800万（持续为正）
- 净利润：1500万、1800万、2000万
- 货币资金：5000万、6000万、7000万
- 有息负债：8000万（基本不变）
- 资产负债率：55%、53%、52%

**框架判断：**

- 经营现金流持续为正且与利润匹配 → 利润质量高
- 货币资金逐年增加 → 资金沉淀健康
- 有息负债不增加但现金流改善 → 自我偿债能力增强
- 经营现金流 / 有息负债 ≈ 35% → 偿债能力有余

**结论：**
稳健型制造业客户，主营业务造血能力强，债务规模稳定。是银行偏好型客户，可进一步了解是否有扩产或供应链金融需求。

## 示例 2：贸易类客户 — 扩张型

**现象：**

- 近三年营收：2亿、3亿、5亿
- 经营现金流：+500万、-300万、-800万（持续恶化）
- 应收账款：3000万、6000万、1.2亿（翻倍增长）
- 短期借款：2000万、4000万、7000万

**框架判断：**

- 营收高速增长但经营现金流恶化 → 扩张是以垫资为代价
- 应收账款增速远高于营收增速 → 回款质量持续恶化
- 短期借款翻倍增长 → 资金链高度依赖融资
- 属于"披着增长外衣的失血型"

**结论：**
需要追问：下游核心客户是谁？账期结构？是否有抵押担保？这类客户如果要做授信，必须要求应收账款抵押或供应链金融结构化安排。

## 示例 3：建筑类客户 — 行业特性型

**现象：**

- 经营现金流：-2000万（年均为负）
- 存货（已完工未结算）：8000万
- 应收账款：1.2亿
- 短期借款：5000万
- 货币资金：1500万

**框架判断：**

- 建筑行业垫资施工是常态，负经营现金流不一定是预警
- 已完工未结算存货能否转化为收入是关键
- 应收账款账龄分布需要穿透
- 短期偿债压力主要看项目回款节点

**结论：**
建筑类客户要单独建判断框架，不能套用一般制造业标准。关键看：甲方付款能力、项目回款节点、是否有明确的结算计划。

## 示例 4：成熟期企业 — 稳定现金牛

**现象：**

- 经营现金流：+8000万
- 投资现金流：-3000万（维持性支出）
- 融资现金流：-4000万（偿还到期贷款）
- 货币资金：1.5亿（稳定）
- 有息负债：0（无贷款）

**框架判断：**

- 经营现金完全覆盖投资和融资支出 → 自给自足
- 主动去杠杆 → 财务风格极度稳健
- 没有融资需求 → 不是当期的信贷目标客户

**结论：**
优质稳定客户，但短期内没有融资需求。可以建立长期关系，关注其上下游供应链金融机会，或等待其新的扩张计划。

## 示例 5：快速判断场景 — 5分钟财报速读

**场景：** 客户经理在拜访前拿到一套简化报表，需要在5分钟内形成判断。

**速读顺序：**

1. 先看经营现金流是否为正（1分钟）
2. 再看有息负债规模和变化趋势（1分钟）
3. 接着看应收账款和存货是否在恶化（2分钟）
4. 最后验证利润与现金流的匹配度（1分钟）

**输出格式（内部记录用）：**

```
客户：[名称]
现金流状态：[正/负/波动]
债务压力：[高/中/低]
资产质量：[健康/需关注/预警]
首要追问：[根据具体情况]
是否值得深入跟进：[是/否]
```

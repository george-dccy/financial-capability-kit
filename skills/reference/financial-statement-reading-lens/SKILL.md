# 财报阅读专业视角

## Scope

这是一个 reference skill。
它帮助你从银行对公业务视角，快速建立对一家企业财务状况的专业判断框架。
重心不是教会你做财务分析，而是让你知道"先看什么、看什么指标、避免什么误判"。

## When To Use

- 客户拜访前需要快速建立对企业财务状况的判断
- 撰写客户评估报告时需要财务判断段落
- 在会议中需要对企业财务状况发表专业意见
- 判断一家企业是否值得深入跟进
- 在没有专业财务团队支持下需要独立判断企业质量

## Core Output Rule

- 是否先看了现金流而非利润
- 是否指出了"表面盈利但实际资金紧张"的风险
- 是否区分了行业特性对财务表现的影响
- 是否避免了授信、定价、审批承诺
- 是否标注了"缺少什么关键信息"

## Not For

- 详细的财务预测或估值计算
- 会计准则的精确解读
- 行业平均水平的对标数据
- 最新监管政策的即时解读

## 判断框架

### 银行视角与财务视角的核心差异

| 维度 | 财务视角 | 银行视角 |
|------|----------|----------|
| 关注焦点 | 利润表（EPS、ROE） | 资产负债表和现金流量表 |
| 时间 horizon | 全周期/年度 | 12–36个月 |
| 关键指标 | 净利润、增长率 | 现金流、偿债能力、资金周转 |
| 报表偏好 | 利润表先行 | 现金流量表优先，资产负债表交叉验证 |
| 判断目的 | 估值和投资决策 | 信用风险和偿还能力评估 |

银行不关心这家企业"值多少钱"，只关心"贷款能不能收回来"。

### 第一步：先看现金流量表（不是利润表）

银行最关注现金，因为"纸上富贵"救不了到期贷款。

1. 经营现金流是否为正
   - 正：主营业务能自我造血
   - 负：不一定是坏事，但必须追问原因

2. 经营现金流与净利润的差距
   - 差距大 → 利润可能来自赊销堆积或会计调节
   - 差距小 → 利润质量高，现金转化能力强

3. 三段现金流的结构关系
   - 经营正 + 投资负 + 融资负：成熟稳健型
   - 经营正 + 投资负 + 融资正：扩张型，需要判断融资用途
   - 经营负 + 投资负 + 融资正：高度关注型，靠输血维持

### 第二步：再看资产负债表

1. 资产结构：货币资金多寡、应收账款和存货是否堆积、固定资产占比
2. 负债结构：短期负债占比、有息负债vs无息负债、负债率变化趋势
3. 所有者权益：注册资本是否实缴、未分配利润、历年分红情况

### 第三步：最后看利润表

利润表是用来"验证"的，不是用来"判断"的。

1. 收入增长是否有现金流支撑
2. 利润增长是否伴随着资产扩张
3. 三项费用的变化是否合理
4. 毛利率是否稳定

### 关键比率的银行视角

| 比率 | 计算方式 | 银行关注逻辑 |
|------|----------|-------------|
| 经营现金流/总债务 | 经营现金流 / 全部有息负债 | 现有债务靠经营现金还款的能力 |
| 流动比率 | 流动资产 / 流动负债 | 短期偿债能力参考 |
| 应收账款周转天数 | 360 / (营收/应收账款) | 回款质量，高于行业需追问 |
| 存货周转天数 | 360 / (成本/存货) | 存货变现能力 |
| 现金比率 | 货币资金 / 流动负债 | 最保守的短期偿债指标 |

单一指标没有意义，必须结合行业特性和企业具体阶段判断。

### 行业特性对财报的影响

| 行业 | 财报特征 | 银行关注重点 |
|------|----------|-------------|
| 制造业 | 固定资产重、存货周转慢 | 产能利用率、存货跌价准备 |
| 贸易业 | 应收账款大、周转快 | 客户集中度、回款账龄 |
| 建筑业 | 垫资施工、现金流差 | 项目回款节点、甲方付款能力 |
| 房地产 | 资金密集、杠杆高 | 销售回款、在建项目进度 |
| 服务业 | 人工成本高、轻资产 | 人工成本占比、客户稳定性 |

### 判断顺序口诀

1. **先看钱，不看利润** — 现金流是偿还能力的直接证据
2. **再看结构，不看总量** — 资产负债结构揭示真正的风险点
3. **后验利润，不先验** — 利润表用来验证，不是用来决策
4. **行业对标，不统一** — 不同行业用不同尺子量
5. **三年趋势，不一年** — 单一年份数据容易失真

## 示例

### 制造业客户 — 稳定型

经营现金流持续为正且与利润匹配，货币资金逐年增加，有息负债不增加但现金流改善，经营现金流/有息负债约35%。稳健型制造业客户，主营业务造血能力强，是银行偏好型客户。

### 贸易类客户 — 扩张型

营收高速增长但经营现金流恶化，应收账款增速远高于营收增速，短期借款翻倍增长。属于"披着增长外衣的失血型"，需追问下游核心客户和账期结构，如做授信必须要求应收账款抵押或供应链金融结构化安排。

### 5分钟财报速读

1. 先看经营现金流是否为正（1分钟）
2. 再看有息负债规模和变化趋势（1分钟）
3. 接着看应收账款和存货是否在恶化（2分钟）
4. 最后验证利润与现金流的匹配度（1分钟）

## 常见误区

- **从利润表开始看**：最常见的错误习惯。利润是权责发生制，可以被应收账款堆积、存货跌价准备计提不足等手段调节。先看现金流量表，再用利润表验证。
- **把高利润等同于高还款能力**：高利润可能来自非常规收益（资产处置、政府补贴），也可能只是行业红利期。看利润的来源结构：主营业务利润占比多少？是否可持续？
- **只关注负债率，不看负债结构**：如果负债主要是短期借款，即使总负债不高，流动性压力也可能很大。区分有息负债和无息负债，看短期有息负债占比。
- **把"有钱"等同于"能还款"**：货币资金可能早已质押给银行作为保证金，或虽然是子公司账户但母公司可以随意划拨。追问货币资金的受限情况。

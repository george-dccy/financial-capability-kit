# 财报阅读专业视角 Reference Skill

## 用途

帮助银行对公业务从业者建立财报阅读的专业判断框架。
重点是"怎么看"，而不是"有哪些财务指标"。

## 适用场景

- 客户拜访前的快速财务画像
- 客户评估报告中的财务判断段落
- 会议中需要对客户财务状况发表意见
- 初筛客户，判断是否值得深入跟进
- 没有专业财务背书时需要独立判断

## 边界

- 这是财务信号的判断框架，不是审计报告
- 不替代审批流程中的评级模型或评分卡
- 不承诺具体的贷款金额或定价
- 需要结合行业、周期、担保等综合判断

## 不负责什么

- 详细的财务预测或估值计算
- 会计准则的精确解读
- 行业平均水平的对标数据
- 最新监管政策的即时解读

# 客户拜访开场框架

## 这是什么

提供客户拜访开场时的专业判断框架：如何选择开场策略、如何建立初始信任、如何快速评估客户状态、如何决定第一个问题的方向。

本 skill 不提供话术文本。话术文本见 `skill.action.market-corporate-client/references/opening-playbook.md`。

## 核心判断问题

1. **冷/热客户**：客户是首次接触还是已有关系？
2. **信任基础**：当前是否有足够的信任支撑深入提问？
3. **时间压力**：客户今天的时间宽裕程度如何？
4. **决策权限**：对面的人是决策者、参与者还是执行者？

## 开场策略选择

| 客户状态 | 推荐开场策略 | 提问方向 |
|----------|-------------|----------|
| 首次接触，低信任 | 价值先示+低承诺 | 影响问题优先 |
| 已有关系，常规拜访 | 进度确认+新发现 | 现状+困难问题 |
| 高信任，深度沟通机会 | 开放探索+共建 | 四类问题均可 |

## 使用边界

- 本 skill = 开场判断框架（what/when/which）
- `opening-playbook.md` = 开场话术模板（具体措辞）
- 两者配合使用，本 skill 先执行判断，再按需调用话术

## 与 opening-playbook 的边界

| 维度 | 本 skill | opening-playbook |
|------|---------|-----------------|
| 焦点 | 判断框架 | 话术模板 |
| 输出 | 策略选择+提问方向 | 具体措辞示例 |
| 使用时机 | 拜访准备/复盘 | 现场执行 |
| 银行针对性 | 强 | 中 |

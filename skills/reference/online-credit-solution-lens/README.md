# 线上授信自动化审批方案视角

## 这个 skill 解决什么

它解决的不是“替你拍板授信”，而是：

- 让你更快判断这类方案该先补什么
- 让“授信方案”章节更有结构
- 让 Agent 在没有足够资料时不乱写结论

## 重点覆盖

- 商业模式与展业场景
- 核心企业或合作平台
- 客群与还款来源
- 数据与核验能力
- 风险缓释
- 授信管理
- 消费者权益保护管理

尤其关注“授信方案”章节：

- 审批模式
- 准入策略
- 贷款额度
- 授信期限
- 贷款期限
- 贷款用途
- 还款方式
- 风险缓释
- 场景价值评估

## 使用边界

- 不替代内部制度
- 不给审批结论
- 不给定价和时效承诺
- 补数不足时必须明确写“待人工确认”或“当前仓库未覆盖”

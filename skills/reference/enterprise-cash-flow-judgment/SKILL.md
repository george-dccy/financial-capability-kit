# 企业现金流判断

## Scope

这是一个 reference skill。
它帮助你从银行信审视角，对企业三张报表的现金流信息做出专业判断框架，而不是套用财务教材的现金流公式。

## When To Use

- 需要从财报中判断企业真实现金生成能力
- 需要判断客户现金流是否健康、是否可持续
- 需要在客户拜访前评估一个企业的现金质量
- 需要判断融资需求是否与现金流状态匹配
- 需要区分"账面利润"和"真实现金"的差距

## Core Output Rule

- 是否区分了"真实现金"与"账面利润"
- 是否指出了现金流信号与融资需求的匹配关系
- 是否标注了还缺什么关键信息
- 是否避免了授信、定价、审批承诺

## Not For

- 详细的财务预测模型
- 现金流折现计算
- 行业对比基准数据
- 最新监管政策的即时解读

## 判断框架

### 银行视角与财务视角的核心差异

| 维度 | 财务视角 | 银行视角 |
|------|----------|----------|
| 关注焦点 | 净利润质量 | 偿还能力 |
| 时间 horizon | 全周期 | 12–24个月 |
| 关键指标 | 净现值、EPS | 经营现金流/债务比、自由现金流 |
| 折旧处理 | 非现金项目 | 还债能力的一部分 |
| 假设前提 | 持续经营假设 | 违约风险下的清偿顺序 |

银行不关心会计利润有多高，只关心：到期的贷款能不能用现金还。

### 第一步：先看经营现金流（Operating Cash Flow）

银行最关注这一段，因为它反映企业主营业务能不能自我造血。

**核心信号判断：**

1. 经营现金流是否为正
   - 正：主营业务能产生现金，基本具备自我偿债能力
   - 负：不一定是坏信号，需结合行业周期和发展阶段判断

2. 经营现金流与净利润的差距
   - 差距大 → 利润质量存疑（可能来自赊销堆积、预收减少）
   - 差距小 → 利润转化为现金的能力强

3. 经营现金流的趋势
   - 三年连续改善 → 经营状态向好
   - 三年连续恶化 → 需要追问原因
   - 波动剧烈 → 周期性行业或业务不稳定

**关键比率（银行关注）：**

- 经营现金流 / 总债务：越高越好，反映以现有债务水平用经营现金还款的能力
- 经营现金流 / 流动负债：判断短期偿债能力
- 经营现金流 / 利息支出：判断利息覆盖程度

### 第二步：再看投资现金流（Investing Cash Flow）

**核心信号判断：**

1. 投资现金流为负的来源
   - 购建固定资产、无形资产：可能是主动扩张信号，也可能是维持性支出
   - 购买理财产品/金融资产：可能是在"钱没地方用"，也可能资金闲置

2. 投资现金流为正的原因
   - 处置固定资产：可能是在收缩或变现
   - 收回投资：可能是前期投资到期回收

3. 与经营现金流的关系
   - 经营为正、投资为负：正常扩张型企业
   - 经营为负、投资为负：观察融资来源，判断是否可持续
   - 经营为正、投资为正：可能是成熟期现金充裕企业

**银行特别关注：**
- 投资支出是否与经营现金流入匹配
- 是否存在大额非主业投资（跨行业并购等）

### 第三步：最后看融资现金流（Financing Cash Flow）

**核心信号判断：**

1. 债务净变化
   - 新增借款：扩张信号，需判断融资用途
   - 偿还债务：财务稳健，但也可能是再融资受阻
   - 债务规模不变：被动状态

2. 股权融资
   - 增资：股东支持信号
   - 分红：利润分配信号，关注是否过度

3. 再融资压力
   - 短期债务到期集中 + 经营现金流不足：预警信号
   - 借新还旧频繁：可能存在资金链紧张

### 三流平衡判断框架

企业资金流（经营）、物资流（供应链）、票据流（结算）是否协调。

**银行判断逻辑：**

1. 钱从哪里来（融资结构）
   - 以经营现金为主 → 自给自足型，风险低
   - 以银行借款为主 → 负债驱动型，关注偿债压力
   - 以股权融资为主 → 资本密集型，关注估值和退出

2. 钱在哪个环节沉淀（资产结构）
   - 现金多、存货少、应收账款少 → 流动性好
   - 存货和应收账款堆积 → 资金被困在经营环节

3. 钱什么时候最紧张（周期节点）
   - 季度末/年末：结算高峰期
   - 春节前：工资和付款集中
   - 行业旺季前：备货占用资金

### 银行现金流预警信号

| 信号 | 含义 | 银行应对 |
|------|------|----------|
| 经营现金流连续两年为负 | 主营业务失血 | 收紧或维持存量，不新增 |
| 应收账款增速远高于营收增速 | 赊销堆积，回款质量差 | 穿透查看账龄分布 |
| 存货突然大幅增加 | 滞销或原材料囤积 | 结合行业判断 |
| 短期借款大幅增加但货币资金不增 | 可能存在资金黑洞 | 追问真实用途 |
| 经营现金流与净利润长期背离 | 利润不实在 | 追问会计政策和业务原因 |
| 关联方往来频繁且金额大 | 可能存在资金占用 | 穿透查看关联方结构 |
| 固定资产购建远超折旧 | 可能存在虚增资产 | 对比产能利用率 |

### 银行现金流判断的核心追求

1. **可验证性**：现金流数字是否有合同、发票、银行流水对应
2. **偿还匹配性**：债务到期时点与现金流入时点是否匹配
3. **趋势可预测性**：未来12–24个月现金状况是否支持偿债
4. **风险传导性**：行业/担保/关联方的风险是否会传导到该企业

## 示例

### 制造业客户 — 经营现金流持续改善

近三年净利润：1000万/1200万/1500万；经营现金流：300万/800万/1400万；货币资金持续增加；银行贷款余额基本持平。

**判断**：经营现金流逐年改善且与利润增长同步，利润质量好；经营现金流/总债务约28%，利息覆盖有余。这是银行偏好的"现金牛"型客户。

### 贸易类客户 — 经营现金流为负但有解释

近三年净利润稳步增长，但经营现金流持续为负；应收账款增速高于营收增速。

**判断**：不是红色警戒，负现金流主要来自应收账款堆积（扩张中的账期管理问题）。需要追问下游核心客户和账期结构，可考虑用供应链金融产品帮助加速回款。

### 建筑类客户 — 融资现金流异常

经营现金流持续为负，新增借款持续增加但主业未扩张，固定资产未增加。

**判断**：高度预警信号。借款用于填补经营亏空，资金链高度依赖银行续贷。必须要求项目回款作为第一还款来源。

## 常见误区

- **把净利润当现金**：净利润是权责发生制，现金流是收付实现制。应收账款堆积可以制造账面利润，但不代表有现金还债。先看经营现金流与净利润的比值，差距过大必须追问原因。
- **只看总量不看结构**：经营现金流为正可能来自压缩供应商付款、拖延工资等短期手段。需要分解经营现金流的来源：销售回款、应收变化、库存变化、应付变化各自贡献多少。
- **把一次性资产处置当作经营改善**：处置土地、房产、股权带来的现金不代表主营业务有造血能力。排除资产处置收益后，看经营现金流是否仍然为正且可持续。
- **忽视短期债务到期集中度**：只关注长期债务安全，忽视短期流动性。"借新还旧"在银行收紧时会导致资金链断裂。做12个月现金流缺口分析，检查短期债务到期是否与经营现金回笼匹配。

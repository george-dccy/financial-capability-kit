# 行业分析基础 Reference Skill

## 用途

从银行对公业务视角，对企业客户所处行业建立快速判断框架。
判断重心不是"这个行业有没有投资机会"，而是"这个行业的企业在信用周期中处于什么位置、有什么风险特征"。

## 适用场景

- 客户拜访前的行业信用风险预判
- 信审报告中的行业分析段落
- 贷后管理中的行业周期跟踪
- 产品匹配前的行业背景判断

## 边界

- 这是行业分析框架的银行视角，不是行业知识百科
- 不替代具体行业的深度研究报告
- 不承诺具体的贷款金额或定价
- 需要结合企业实际经营情况综合判断

## 不负责什么

- 具体行业（水泥、光伏、房地产等）的深度知识
- 行业政策最新即时解读
- 投资建议或估值判断
- 具体的授信审批结论

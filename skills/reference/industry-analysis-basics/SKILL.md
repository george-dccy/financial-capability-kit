# 行业分析基础

## Scope

这是一个 reference skill。
它帮助你从银行对公业务视角，快速建立接触企业客户前或贷后管理中的行业分析判断框架。
重心不是教会你做行业研究，而是让你知道"先看什么行业维度、看什么信号、避免什么误判"。

## When To Use

- 客户拜访前需要快速建立对客户所处行业的信用风险判断
- 信审报告中的行业分析段落
- 贷后管理中需要判断行业周期变化对企业的影响
- 判断一家企业所在行业是否值得深入合作
- 在没有专业行业研究团队支持下需要独立判断行业信用特征

## Core Output Rule

- 是否从银行信用风险视角而非投资视角切入
- 是否有判断优先级（首次接触看什么、行业分化看什么、贷后管理看什么）
- 是否避免了具体行业知识堆砌
- 是否标注了"缺少什么关键信息"
- 是否避免了授信、定价、审批承诺

## Not For

- 具体行业（水泥、光伏、房地产等）的深度知识
- 行业政策最新即时解读
- 投资建议或估值判断
- 具体的授信审批结论

## 判断框架

### 银行视角与行业研究视角的核心差异

| 维度 | 行业研究视角 | 银行视角 |
|------|-------------|----------|
| 关注焦点 | 增长潜力、竞争格局 | 信用风险、还款来源、周期敏感性 |
| 时间 horizon | 3–10年 | 12–36个月 |
| 关键问题 | 行业天花板在哪 | 行业下行时企业能不能还贷 |
| 核心框架 | 波特五力、产业链、生命周期 | 生命周期位置、周期敏感度、监管影响 |
| 判断目的 | 投资机会 | 信用风险和偿债能力 |

银行不关心这个行业"有没有投资机会"，只关心"这个行业的企业贷款能不能收回来"。

### 第一步：判断行业生命周期位置

1. **导入期（Embryonic）**：技术不成熟、企业普遍亏损。银行风险极高，原则上不介入。
2. **成长期（Growth）**：市场需求快速扩大、竞争格局未定。银行风险中等，优选龙头。
3. **成熟期（Mature）**：市场需求稳定、竞争格局清晰。银行风险相对可控，适合中期流动资金贷款。
4. **衰退期（Decline）**：市场需求萎缩。银行风险高，压缩存量，不新增。

**关键信号**：行业增速 vs. GDP 增速（持续高于3倍以上为成长期）、行业利润率趋势、新进入者数量。

### 第二步：判断波特五力中的信用风险权重

| 波特五力 | 银行关注权重 | 核心判断问题 |
|---------|------------|-------------|
| 行业内现有竞争者 | 最高 | 竞争是否导致价格战，压垮企业现金流？ |
| 新进入者威胁 | 中 | 壁垒够不够高？新进入者会不会分走还款来源？ |
| 上游议价能力 | 中 | 上游是否过强，导致企业资金被占用？ |
| 下游议价能力 | 中 | 下游是否过强，导致企业收款被拖延？ |
| 替代品威胁 | 低 | 替代品是否在改变需求结构？ |

银行核心关注：竞争强度是否侵蚀企业现金流。

### 第三步：判断行业对监管政策的敏感度

**高敏感行业**：房地产（限购/限贷/预售资金监管）、地方政府融资平台（隐性债务清理）、教培（"双减"政策）、化石能源（碳中和政策）。

**中等敏感行业**：制造业（环保/安全生产）、医药（带量采购）、食品（食品安全监管）。

**低敏感行业**：商业银行本身、必需消费品、部分服务业。

### 第四步：判断行业周期性位置

1. 观察行业核心指标的历史同比变化（营收增速、利润率、存货水平）
2. 判断是供给周期（产能出清慢，如钢铁、煤炭）还是需求周期（恢复快，如汽车、家电）
3. 判断当前位置：复苏期 → 繁荣期 → 滞胀期 → 衰退期

### 快速判断优先级

**首次接触客户**：行业名称判断生命周期 → 监管敏感度判断政策风险 → 行业平均利润率判断盈利空间。

**行业分化明显时**：市场份额趋势、产业链位置、固定资产占比、现金转化周期。

**贷后管理**：行业周期信号（领先指标是否出现拐点）、政策变化信号、竞争格局信号、上下游信号。

## 示例

### 首次接触制造类企业

汽车零部件制造，年营收约3亿元。生命周期判断：整体成熟期，但零部件国产替代是结构性机会。波特五力：下游整车厂议价能力极强，账期长。银行关注点：下游整车厂账期是否合理，应收账款是否过度集中。

**输出**：行业信用风险中等，核心风险点为下游回款周期和是否进入新能源供应链，验证前五大客户集中度和账龄分布。

### 贷后管理中发现行业预警信号

工程建筑企业新中标合同金额同比下降40%。建筑业受房地产下行和基建投资周期双重影响，当前处于行业下行期。行业内竞争加剧，下游甲方议价能力增强。

**输出**：行业周期位置为下行期，主要风险为房地产下行导致需求萎缩，建议关注应收账款回收速度和垫资施工风险。

### 判断新兴行业是否可介入

新能源充电桩运营企业申请流动资金贷款。行业处于导入期向成长期过渡，商业模式尚未完全验证，依赖补贴。监管敏感性极高，补贴退坡直接影响盈利模型。

**输出**：行业信用风险高，现阶段原则上不介入；如政策要求介入，需强担保结构加限制性条款。

## 常见误区

- **把行业增长当企业信用**：高增长行业往往竞争激烈，淘汰率高。"行业好"不代表"企业好"，先看企业在行业中的市场份额趋势，再看行业增速。
- **用投资视角替代信用视角**：投资视角关注增长潜力和估值，信用视角关注现金流和偿债能力。有投资价值的行业也可能有大面积违约风险。
- **忽视行业周期位置**：用行业繁荣期的数据判断企业在下行期的偿还能力会高估抗风险能力。至少看一个完整周期（3-5年）的数据。
- **把政策出台当成风险消除**：政策从出台到落地有时间差，效果不一定达预期。看企业实际经营数据是否因政策而改善，而不是假设政策效果自动传导。

# Frameworks

## 一、银行视角行业分析优先顺序

### 第一步：判断行业生命周期位置

银行最关注行业所处阶段，因为这决定了信用风险的基本特征。

**核心判断框架（参考 Porter 生命周期模型 + CFA Industry Analysis）：**

1. 导入期（Embryonic）
   - 特点：技术不成熟、市场认可度低、企业普遍亏损
   - 银行风险：极高，业务模式未验证，违约概率大
   - 常见行业：早期新能源、尖端生物医药、AI 应用早期
   - 银行应对：原则上不介入，政策要求例外需强担保

2. 成长期（Growth）
   - 特点：市场需求快速扩大、竞争格局未定、技术路线收敛
   - 银行风险：中等，收入增长快但竞争淘汰率高
   - 常见行业：新能源汽车渗透率提升期、部分新材料、部分消费品
   - 银行应对：优选龙头，关注市场份额稳定性，慎介入技术路线未定型行业

3. 成熟期（Mature）
   - 特点：市场需求稳定、竞争格局清晰、行业增速与 GDP 增速趋同
   - 银行风险：相对可控，头部企业现金流稳定
   - 常见行业：传统制造业中的头部企业、商业银行、消费品龙头
   - 银行应对：适合中期流动资金贷款，优选行业前三名

4. 衰退期（Decline）
   - 特点：市场需求萎缩、技术或消费者偏好转移
   - 银行风险：高，现有债务偿还依赖资产处置或转型
   - 常见行业：传统燃油车、部分高耗能行业、部分线下零售
   - 银行应对：压缩存量，不新增，担保条件从严

**关键信号：**
- 行业增速 vs. GDP 增速：持续高于 GDP 3 倍以上 → 成长期
- 行业利润率趋势：是否向行业平均收敛
- 新进入者数量：是否在增加（成长期）还是在减少（成熟/衰退）

---

### 第二步：判断波特五力中的信用风险权重

Michael Porter 提出的五力框架，银行视角的权重排序与投资视角不同：

| 波特五力 | 银行关注权重 | 核心判断问题 |
|---------|------------|-------------|
| 行业内现有竞争者 | ★★★（最高） | 竞争是否导致价格战，压垮企业现金流？ |
| 新进入者威胁 | ★★ | 壁垒够不够高？新进入者会不会分走还款来源？ |
| 上游议价能力 | ★★ | 上游是否过强，导致企业资金被占用？ |
| 下游议价能力 | ★★ | 下游是否过强，导致企业收款被拖延？ |
| 替代品威胁 | ★ | 替代品是否在改变需求结构？ |

**银行核心关注：竞争强度是否侵蚀企业现金流**

行业内价格战 → 毛利率压缩 → 经营现金流减少 → 偿债能力下降

---

### 第三步：判断行业对监管政策的敏感度

银行天然关注监管政策，因为政策变化直接影响还款来源的稳定性。

**高敏感行业（政策直接决定信用风险）：**

| 行业 | 关键监管变量 | 银行关注维度 |
|------|------------|-------------|
| 房地产 | 限购、限贷、预售资金监管 | 销售回款是否被锁定 |
| 地方政府融资平台 | 隐性债务清理、城投政策 | 地方政府财力变化 |
| 教培 | "双减"政策 | 行业存续性 |
| 互联网金融 | 金融科技监管 | 业务模式是否合规 |
| 化石能源 | 碳中和政策 | 转型成本有多高 |
| 光伏/风电 | 补贴退坡、电价政策 | 装机意愿 |

**中等敏感行业（政策间接影响）：**
- 制造业：环保政策、安全生产政策
- 医药：带量采购、医保谈判
- 食品：食品安全监管

**低敏感行业：**
- 商业银行本身
- 必需消费品（食品饮料基础品类）
- 部分服务业

---

### 第四步：判断行业周期性位置

行业周期影响企业现金流的稳定性，银行需要判断当前处于周期的哪个位置。

**周期判断优先顺序：**

1. 观察行业核心指标的历史同比变化
   - 营收增速、利润率、存货水平
   - 与历史均值对比：偏高/偏低/接近均值

2. 判断是供给周期还是需求周期
   - 供给周期（产能出清慢，下行期长）：钢铁、煤炭、光伏
   - 需求周期（政策/消费驱动，恢复快）：汽车、家电

3. 判断当前位置
   - 复苏期：需求回升、产能利用率提升
   - 繁荣期：高产能利用率、价格上行
   - 滞胀期：成本上升挤压利润
   - 衰退期：需求萎缩、产能过剩

## 二、快速判断优先级

### 首次接触客户：看什么

1. 行业名称 → 判断生命周期大致位置
2. 行业监管敏感度 → 判断政策风险大小
3. 行业平均利润率 → 判断企业是否有正常盈利空间

### 行业分化明显时：看什么

当同一行业出现明显分化（头部 vs. 中小），银行需要判断：

1. 市场份额趋势：是企业份额在扩大还是在收缩
2. 产业链位置：上游、中游、下游，议价能力不同
3. 固定资产占比：重资产 vs. 轻资产，退出成本不同
4. 现金转化周期：行业特有的资金占用特征

### 贷后管理：看什么

1. 行业周期信号：领先指标是否出现拐点
2. 政策变化信号：监管文件是否密集出台
3. 竞争格局信号：是否有新进入者或价格战迹象
4. 上下游信号：核心客户/供应商是否出现异常

---

## 三、银行行业分析的核心输出

行业分析完成后，银行应形成以下判断：

1. **行业信用风险评级**：低/中/高
2. **周期位置**：复苏/繁荣/滞胀/衰退
3. **主要风险点**：竞争/政策/周期性/技术/资金占用
4. **需要进一步验证的事项**：企业与行业的差异点
5. **授信策略建议**：谨慎/维持/优化/压缩（不是审批结论，是分析输入）

---

## 四、使用边界

- 行业分析是信用判断的重要输入，不是审批替代
- 行业判断不能替代对企业个体的具体分析
- 行业周期位置预判本身有不确定性，不能作为精确预测
- 政策判断依赖公开信息，不包含内部监管指引
- 需要结合企业实际控制人、担保情况、征信记录综合判断

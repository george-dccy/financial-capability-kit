# Examples

以下示例展示银行视角行业分析的判断输出。

---

## 示例一：首次接触制造类企业

**场景：** 对公客户经理首次拜访一家做汽车零部件的中小制造企业。

**输入：** 行业名称"汽车零部件制造"，企业年营收约 3 亿元。

**判断过程：**

1. 生命周期判断
   - 汽车零部件行业整体处于成熟期
   - 零部件国产替代是结构性机会，部分细分子行业处于成长期
   - 判断：该企业所在子行业需要进一步了解

2. 波特五力判断
   - 上游：大宗原材料（钢材、铝），议价能力中等
   - 下游：整车厂，议价能力极强，账期长
   - 行业内：竞争激烈，呈现"整车厂绑定"格局
   - 银行关注点：下游整车厂账期是否合理，应收账款是否过度集中

3. 监管敏感性
   - 汽车零部件行业受新能源转型政策影响
   - 判断：该企业是否已进入新能源供应链

**输出（银行视角）：**
- 行业信用风险：中（成熟行业，但需判断子行业和客户结构）
- 核心风险点：下游回款周期、是否进入新能源供应链
- 验证事项：前五大客户集中度、账龄分布、是否有 tier-1 整车厂资质

## 示例二：贷后管理中发现行业预警信号

**场景：** 客户经理在贷后管理中发现一家工程建筑企业的新中标合同金额同比下降 40%。

**判断过程：**

1. 行业周期信号评估
   - 建筑业受房地产下行和基建投资周期双重影响
   - 当前处于行业下行期，景气度低迷
   - 判断：该客户中标量下降符合行业周期特征

2. 波特五力快速评估
   - 行业内竞争加剧：房地产下行期，建筑企业抢单激烈
   - 下游甲方议价能力增强：房地产开发商资金紧张，付款条件恶化
   - 上游材料：钢材价格相对稳定，议价能力中等

3. 政策信号
   - 房地产"三条红线"持续压制开发商杠杆
   - 城投融资政策收紧

**输出（银行视角）：**
- 行业周期位置：下行期（衰退偏后段）
- 主要风险：房地产下行导致建筑业需求萎缩
- 企业层面的风险：中标量下降是需求萎缩还是竞争力下降
- 建议：关注应收账款回收速度，评估是否存在垫资施工风险

## 示例三：判断新兴行业是否可介入

**场景：** 某新能源充电桩运营企业申请流动资金贷款。

**判断过程：**

1. 生命周期判断
   - 充电桩运营行业整体处于导入期向成长期过渡
   - 商业模式尚未完全验证，依赖补贴和政策
   - 行业整体盈利能力弱，运营企业普遍亏损

2. 波特五力判断
   - 上游：设备制造，国内竞争激烈，议价能力低
   - 下游：电动车车主，议价能力中等，依赖流量
   - 新进入者威胁：高，技术壁垒不高，资本壁垒中等
   - 替代品：换电站、慢充等

3. 监管敏感性
   - 极高。充电桩建设受新能源政策、城市规划、土地使用政策多重影响
   - 补贴退坡直接影响盈利模型

**输出（银行视角）：**
- 行业信用风险：高（导入期，商业模式未成熟）
- 核心风险：补贴退坡、运营效率、资产利用率
- 建议：现阶段原则上不介入；如政策要求介入，需强担保结构 + 限制性条款（最低利用率承诺）
